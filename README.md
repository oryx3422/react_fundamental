# React Fundamental

Проект выполнен на **React + JavaScript**.

Веб-приложение для работы с постами: отображение списка постов, создание новых постов, фильтрация, сортировка, поиск и детальный просмотр. Проект демонстрирует основные концепции React: управление состоянием, жизненный цикл компонентов, работа с хуками, роутинг и работа с API.

## Функционал

- **Работа с API**: получение данных с JSONPlaceholder API
- **Отображение списка постов**: список постов с возможностью удаления
- **Создание постов**: создание новых постов через модальное окно
- **Страница деталей**: подробная информация по конкретному посту с комментариями
- **Фильтрация и сортировка**: фильтрация постов по различным параметрам
- **Поиск**: поиск постов по содержимому
- **Пагинация**: постраничная навигация по постам
- **Бесконечная лента**: автоматическая подгрузка постов при прокрутке
- **Роутинг**: навигация между страницами с защитой приватных маршрутов
- **Аутентификация**: простая система входа в приложение
- **Обработка ошибок**: обработка ошибок при загрузке данных
- **Кастомные хуки**: переиспользуемый функционал через хуки
- **UI компоненты**: библиотека переиспользуемых UI компонентов

## Инструкция по запуску проекта

### Требования

- Node.js v14 или выше
- npm или yarn

### Установка и запуск

#### 1. Клонирование репозитория

```bash
git clone https://github.com/oryx3422/react_fundamental
cd react_fundament
```

#### 2. Установка зависимостей

```bash
npm install
```

#### 3. Запуск приложения

```bash
npm start
```


#### 4. Сборка для продакшена

```bash
npm run build
```

## Используемые технологии

- **React** - библиотека для создания пользовательских интерфейсов
- **React Router DOM** - маршрутизация в React приложении
- **Axios** - HTTP клиент для работы с API
- **CSS Modules** - модульные стили

## Основные концепции

- Управление состоянием компонентов
- Жизненный цикл компонентов
- React Hooks (useState, useEffect, useContext, useRef)
- Кастомные хуки
- React Context для глобального состояния
- Роутинг и защита маршрутов
- Работа с API (async/await)
- Обработка ошибок
- Оптимизация производительности
